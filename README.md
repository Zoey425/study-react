#리액트 스터디 가위바위보
가위바위보 게임을 만들어 봅시다.

    유저스토리
    1. 유저는 박스 두개를 볼 수 있다.
    2. 유저는 박스하단에 가위바위보 버튼을 볼 수 있다.
    3. 버튼을 클릭하면 클릭한 아이템이 유저박스에 보인다.
    4. 버튼을 클릭하면 컴퓨터 아이템은 랜덤하게 선택이 된다.
    5. 3,4번 아이템을 가지고 누가 이겼는지 승패를 나눈다.
    6. 박스 테두리가 결과에 따라 색이 변한다. 지면 빨강, 이기면 녹색, 비기면 검정
